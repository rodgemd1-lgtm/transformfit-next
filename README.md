# TransformFit Next — README

> A fitness coaching app that reads the week you are actually having.
> Rebuilt from scratch with research-backed design. MiroFish validated. Fogg optimized.

## Quick Start

```bash
npm install
npm run dev
# → http://localhost:3000
```

## Stack

- **Next.js 15** (App Router, Turbopack)
- **React 19**
- **TypeScript** (strict)
- **Tailwind CSS v4**
- **Framer Motion** (scroll-triggered animations)
- **Radix UI** (accessible primitives)

## Design System

Ink (#0A0A0A) + Paper (#F0EDE8) + Orange (#F97316) + Emerald (#10B981)
Inter + Playfair Display
WCAG 2.1 AA compliant

## Pages

- `/` — Landing page (Clinical variant, 20+ sections)
- `/dashboard` — Demo dashboard (no-signup-wall proof)
